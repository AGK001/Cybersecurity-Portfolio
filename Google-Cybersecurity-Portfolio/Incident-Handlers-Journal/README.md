## üîç Featured Incident Logs

### 1. Ransomware Attack (Healthcare Sector)
* **Scenario:** An organized group deployed ransomware via a phishing email at a U.S. healthcare clinic.
* **Impact:** Critical patient files were encrypted, disrupting healthcare operations.
* **Key Insight:** Highlighted the urgent need for email filters, regular backups, and employee security awareness training.

### 2. ICMP Flood (DDoS) Response
* **Scenario:** A two-hour DDoS attack traced to an unconfigured firewall.
* **Impact:** Internal network services were rendered unresponsive.
* **Remediation:** Applied the NIST CSF to implement ICMP rate-limiting and source IP verification.

### 3. Mail Server Brute-Force Investigation
* **Scenario:** Persistent login attempts identified around the same time each day targeting a mail server.
* **Tools Used:** Analyzed authentication logs and network traffic.
* **Remediation:** Recommended strict firewall rules to restrict mail server access to authorized IP addresses only.
